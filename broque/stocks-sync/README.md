# fl-stocks-sync

Synchronizes end of day stocks information from various sources

## Primary source

### Twelve data

Stocks are loaded from twelve data, limited to prioritized stocks due to credit limits for basic subscription.

#### S&P 500

List of prioritized EOD stocks are in `s&p500-consituents.json`. Should be updated whenever this [information](https://datahub.io/core/s-and-p-500-companies#constituents) changes.

#### Remaining US stocks

With any remaining credits, other stock information should be obtained at random and updated.
